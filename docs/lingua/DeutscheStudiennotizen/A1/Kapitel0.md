# Lektion 2 元音和元音长短音规则

# Vokal

## 德语元音 Aa Ii Uu Ee Oo

所有德语元音都有长短音

### 长音规则

- 以元音结尾的单词，该元音读长音
- 元音后只有一个辅音，该元音读长音
- 相同两个元音叠加，该元音读长音
- 元音后有h，h不发音，该元音读长音

### 短音规则

- 元音后有两个及其以上的辅音，该元音读短音
- 除了长音规则，剩下的基本都是短音
- 特殊情况 an ab hat
